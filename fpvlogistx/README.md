## 🌟 О проекте

**FPVlogistX** — это современный одностраничный сайт для компании, занимающейся логистикой и доставкой посылок с использованием FPV-дронов. Сайт разработан с акцентом на скорость, инновационность и удобство использования.

### 🚀 Особенности

- ✨ Современный дизайн с эффектом стекла (glassmorphism)
- 📱 Полностью адаптивный интерфейс (мобильные, планшеты, десктоп)
- 💰 Интерактивный калькулятор стоимости доставки
- 🎨 Плавные анимации при скролле
- 🎯 Оптимизирован для SEO
- ⚡ Быстрая загрузка

### 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6+)
- Font Awesome иконки
- Google Fonts
